- 防抖debounce(事件被触发n秒后执行回调函数, 如果在这n秒内又被触发, 则重新计时)
<Card>
  ## 场景
  - 按钮防止重复点击
  - 搜索框连续输入的情况等输入停止后再搜索
  - 浏览器窗口大小变化仅需要出发一次事件
</Card>
```js
//debounce
let timer = null;

function debounce(fn, wait, ...arg) {
    if(timer) {
        clearTimeout(timer);
        timer = null;
    }
    timer = setTimeout(() => {
        fn.apply(this, ...arg);
    }, wait)
}
```
- 节流
```js
//throttle
let last = 0;

function throttle(fn, gapTime, ...arg) {
    let nowTime = Date.now();
    if(nowTime - last > gapTime) {
        fn.apply(this, ...arg);
        last = nowTime;
    }
}
```
