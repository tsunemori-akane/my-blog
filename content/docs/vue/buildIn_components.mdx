## Transition
- 使用`transition`组件使路由切换过程中页面呈现淡出淡入效果
```vue
<template>
  <div id="app">
    <transition name="fade" mode="out-in">
      <router-view />
    </transition>
  </div>
</template>
<style>
 .fade-enter {
    opacity: 0;
 }
 .fade-enter-active,
 .fade-leave-active {
    transition: opacity 0.5s ease-out
 }
 .fade-leave-to {
  opacity: 0;
 }
</style>
```
要实现更加复杂的效果, 可以使用[javascript hooks](https://vuejs.org/guide/built-ins/transition.html#javascript-hooks)
